# Deploy Agent Studio for FREE

**Heroku no longer has a free tier** (Eco dynos start around $5/month). For zero-cost hosting, use **Render** (recommended) or the alternatives below.

## Render (recommended — $0/month)

| | |
|---|---|
| **Cost** | Free |
| **Catch** | App sleeps after ~15 min idle; first request may take 30–60s to wake |
| **Storage** | Ephemeral — SQLite resets on redeploy (same as Heroku) |

### Steps (about 10 minutes)

1. **Push your code to GitHub**  
   The `agent` folder must be in a GitHub repository.

2. **Sign up** at [render.com](https://render.com) (GitHub login works).

3. **New → Blueprint** → connect your repo → Render reads `render.yaml`.

4. **Set required env vars** in the Render dashboard (Environment):

   | Variable | Example |
   |----------|---------|
   | `APP_BASE_URL` | `https://agent-studio-xxxx.onrender.com` |
   | `OAUTH_REDIRECT_URI` | `https://agent-studio-xxxx.onrender.com/api/gmail/callback` |
   | `GOOGLE_CLIENT_SECRETS_JSON` | Full OAuth JSON (one line) |

   `APP_SECRET` is auto-generated by the blueprint.

5. **Deploy** → wait for build → open your URL.

6. **Google Cloud Console** → add the same redirect URI under OAuth credentials.

7. **Verify:** `https://your-app.onrender.com/api/health` → `{"status":"ok"}`

### Manual deploy (without Blueprint)

1. Render → **New Web Service**
2. Connect GitHub repo
3. **Environment:** Docker
4. **Plan:** Free
5. **Health check path:** `/api/health`
6. Add env vars as above

## Free tier limitations (all platforms)

- **No persistent disk** — user accounts and SQLite are lost on redeploy/restart unless you add Postgres + object storage later.
- **Cold starts** — idle apps sleep; first visit is slow.
- **HTTPS required** — Gmail OAuth needs `https://` URLs in `APP_BASE_URL` and Google Console.

## Optional: OpenAI / Twilio / SMTP

Set in Render env vars when you want live agent actions (otherwise agents simulate):

```env
OPENAI_API_KEY=sk-...
TWILIO_ACCOUNT_SID=...
TWILIO_AUTH_TOKEN=...
TWILIO_FROM_NUMBER=+1...
SMTP_HOST=smtp.gmail.com
SMTP_USER=...
SMTP_PASSWORD=...
SMTP_FROM=...
```

## Upgrade path

When you outgrow free tier:

1. **Render paid** ($7+/mo) — always on, no sleep
2. **Heroku Eco** — see [HEROKU.md](HEROKU.md) if you prefer Heroku + GitHub Actions CI/CD
3. **Postgres add-on** — durable users/sessions (replace SQLite)
